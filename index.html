    未登录
    • 退出所有设备
      撤销你在所有手机、平板、电脑上的登录会话，下次都需要重新登录。
    • 导出书单备份
      下载 JSON 文件，包含书单 / 阅读记录 / 摘抄 / 对话历史 / 跨书关联。可以用「导入数据」恢复，或迁移到新账号。
    • 完整账号导出（GDPR/PIPL 合规）
      在「导出书单备份」基础上额外包含：AI 调用日志、Agent 执行轨迹、上传图片清单。供你行使数据知情权。
    • 导入数据
      从「导出书单备份」生成的 .json 文件恢复。会替换当前账号的全部内容。
    • 从 Excel 批量加书
      从 .xlsx 表格批量添加书单。表格首行需含「书名」列（可选：作者 / 状态 / 标签 / 总页数 / 开始时间 / 完成时间 / 译者 / 简介 / 喜欢程度）。已存在的书不会重复。
    • 注销账号
      永久删除你的全部数据，包括上传图片、聊天记录、服务端日志。不可恢复。
